# LayerNorm 的归一化维度详解

## 答案

**LayerNorm 对每个词的 embedding 维度（最后一维）归一化，不跨词、不跨 batch。**

```
输入: [Batch, seq_len, emb_dim] = [2, 1024, 768]

LayerNorm 对每个位置的 768 维向量独立归一化
         ↓
输出: [2, 1024, 768]  （维度不变）
```

---

## 可视化

```
输入张量 [2, 4, 768]:

Batch 0:
┌─────────────────────────────────────────────────┐
│ 位置0: [0.1, 0.2, ..., 0.3]  ← 对这 768 维归一化 │
│ 位置1: [0.4, 0.5, ..., 0.6]  ← 对这 768 维归一化 │
│ 位置2: [0.7, 0.8, ..., 0.9]  ← 对这 768 维归一化 │
│ 位置3: [1.0, 1.1, ..., 1.2]  ← 对这 768 维归一化 │
└─────────────────────────────────────────────────┘

Batch 1:
┌─────────────────────────────────────────────────┐
│ 位置0: [...]  ← 独立归一化                       │
│ 位置1: [...]  ← 独立归一化                       │
│ 位置2: [...]  ← 独立归一化                       │
│ 位置3: [...]  ← 独立归一化                       │
└─────────────────────────────────────────────────┘

每个位置的 768 维向量独立计算 mean 和 std
共有 2 × 4 = 8 次独立的归一化操作
```

---

## 数学公式

对于位置 $(b, t)$ 的向量 $x \in \mathbb{R}^{768}$：

$$
\mu = \frac{1}{768} \sum_{i=1}^{768} x_i
$$

$$
\sigma = \sqrt{\frac{1}{768} \sum_{i=1}^{768} (x_i - \mu)^2 + \epsilon}
$$

$$
\hat{x}_i = \frac{x_i - \mu}{\sigma}
$$

$$
y_i = \gamma \cdot \hat{x}_i + \beta
$$

**关键**：$\mu$ 和 $\sigma$ 是对 **768 维**计算的，不是对 batch 或 seq_len 计算的。

---

## 对比：LayerNorm vs BatchNorm

| | LayerNorm | BatchNorm |
|--|-----------|-----------|
| **归一化维度** | 最后一维（emb_dim=768） | batch 维度 |
| **统计量计算** | 每个样本每个位置独立 | 整个 batch 共享 |
| **归一化次数** | batch × seq_len 次 | emb_dim 次 |
| **用于** | Transformer | CNN |

```
LayerNorm 归一化方向:
┌─────────────────────────────────────┐
│ [b, t, :] → 对 emb_dim 维度归一化    │
│            ←─────────────────→       │
│                 768 维               │
└─────────────────────────────────────┘

BatchNorm 归一化方向（如果用的话）:
┌─────┐
│ ↓   │  对 batch 维度归一化
│ ↓   │
│ ↓   │
└─────┘
```

## 代码验证

```python
import torch
import torch.nn as nn

x = torch.randn(2, 4, 768)  # [batch, seq_len, emb_dim]
ln = nn.LayerNorm(768)       # 只传入最后一维的大小

# LayerNorm 对每个 [768] 向量独立归一化
output = ln(x)  # [2, 4, 768]

# 验证：每个位置的输出均值≈0，标准差≈1
print(output[0, 0, :].mean())  # ≈ 0
print(output[0, 0, :].std())   # ≈ 1
```
